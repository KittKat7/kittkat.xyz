Nebula Assistant

AppHack2024 contest: Second Place Winner!

Nebula Assistant is a virtual assistant which uses the StackOverflow API to find answers to any tech or programming question.

Features

StackOverflow Search

Speach To Text Input

Text To Speach Output

CLI Mode (no TTS or STT)

Weather Info

Telling Jokes

Dictionary Definitions

Urban Dictionary Definitions

News Headlines

Launch YouTube Videos

About

Original Plan

This project was originally designed for the App State AppHack2024 event, which is a 24 hour programming competition. The original plan was to make a virtual assistant similar to Google Assistant, but instead of simply doing a Google search, searching StackOverflow. Doing this would serve to provide results more useful to tech people and programmers.
